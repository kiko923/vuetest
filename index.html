⚡CDNJS API

轻量、快速的边缘 API 展示页 · 构建于 Edge Functions ✨

📚 可用接口 Read-only

  • 获取库信息：
    GET /api/libraries?name={name}&version={optional}
  • 获取库信息（POST）：JSON 参数
    POST /api/libraries · { name, version? }
  • EdgeOne 指标示例：
    GET /api/num · 可选 ?act=flow
  • 同步至 COS：JSON 参数
    POST /api/sync · { name, version, key }

提示：路由大小写敏感，`/api/libraries` 与目录 `edge-functions/api/libraries.js` 一一对应。

🧩 示例

Examples
GET /api/libraries?name=jquery
GET /api/libraries?name=vue&version=3.5.0
GET /api/num
GET /api/num?act=flow

POST /api/libraries
Body: { "name": "jquery", "version": "3.7.1" }

POST /api/sync
Body: { "name": "jquery", "version": "3.7.1", "key": "jquery.min.js" }
